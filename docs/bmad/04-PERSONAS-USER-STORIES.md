### 3.1 Epic 1 : Configuration Initiale

```
┌─────────────────────────────────────────────────────────────────────────────┐
│  EPIC 1 : ONBOARDING                                                         │
│  Objectif : Permettre à l'utilisateur de configurer son profil rapidement   │
└─────────────────────────────────────────────────────────────────────────────┘
```

| ID | User Story | Priorité | Statut |
|----|-----------|----------|--------|
| US-1.1 | En tant que nouvel utilisateur, je veux sélectionner mon cycle de travail (2/2/3, 4/2, hebdo) pour que l'app calcule correctement mes jours | Must | ✅ Done |
| US-1.2 | En tant que nouvel utilisateur, je veux entrer la date de début de mon cycle A pour que le calendrier affiche les bonnes semaines | Must | ✅ Done |
| US-1.3 | En tant que nouvel utilisateur, je veux saisir mes soldes actuels de chaque compteur pour partir de ma situation réelle | Must | ✅ Done |
| US-1.4 | En tant que nouvel utilisateur, je veux définir mon objectif CET pour que l'app me guide vers cet objectif | Should | ✅ Done |
| US-1.5 | En tant qu'utilisateur, je veux pouvoir modifier ma configuration à tout moment pour m'adapter aux changements | Should | ✅ Done |

| Critère | Cible |
|---------|-------|
| Temps chargement initial | <2s |
| Temps génération combinaisons | <500ms |
| Temps réponse UI | <100ms |

### 6.2 Accessibilité

| Critère | Cible |
|---------|-------|
| Score Lighthouse a11y | >90 |
| Navigation clavier | 100% |
| Screen reader compatible | NVDA, VoiceOver |

### 6.3 Utilisabilité

| Critère | Cible |
|---------|-------|
| Tâches complétables sans aide | >95% |
| Erreurs utilisateur par session | <1 |
| SUS (System Usability Scale) | >75 |
